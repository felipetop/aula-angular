Criando uma Nova Branch a partir de um Commit

Abra o terminal e navegue até o diretório do seu projeto.

Use o seguinte comando para criar e mudar para uma nova branch a partir do commit desejado (substitua COMMIT_HASH pelo hash do commit e new-branch-name pelo nome que deseja dar à nova branch):

git checkout -b new-branch-name COMMIT_HASH

Por exemplo, para criar uma nova branch a partir do commit do tópico "Router", você usaria:

git checkout -b feature/router aba117a542338cf2b1c49c3b510532f778dfec68

Você agora estará na nova branch e poderá continuar trabalhando a partir desse ponto no histórico.

Como gerar um novo componente com módulo?

Utilizaremos o lazy loading, logo cada módulo irá importar os itens individualmente, deixando o código mais organizado e performático.

ng generate module name --routing
ng generate component name
